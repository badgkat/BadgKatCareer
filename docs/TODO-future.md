# Future Ideas

Ideas that came up during design but are out of scope for the current spec.

## Failure Reactions (ChatterBox Action Mode)
Add `onState = PARAMETER_FAILED` ChatterBox behaviours to active contracts so characters react to failures in real time via the non-pausing Action PiP feed. Gene deadpanning when a rocket explodes, Gus wincing when the rover flips, Wernher treating explosions as data. This is gold but needs the core dialogue system in place first.

## Auto-Accept Mission Tree
Add `autoAccept = true` to milestone and exploration contracts so they feel like a campaign mission list rather than a contract board. One-line change per contract. The ChatterBox dialogues become the mission briefing. Consider pairing with a visual mission tree mod if one exists.

## Action Mode Radio Chatter (Approach C)
Mid-mission ambient callouts via Action presentation mode. Gene radio-calling when you hit orbit, Wernher chiming in on science collection, Gus grunting approval when the rover moves. Needs the core Tier 1/2 dialogues written first to establish character voices.
