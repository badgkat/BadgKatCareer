# Future Ideas

Ideas that came up during design but are out of scope for the current spec.

## Failure Reactions (ChatterBox Action Mode)
Add `onState = PARAMETER_FAILED` ChatterBox behaviours to active contracts so characters react to failures in real time via the non-pausing Action PiP feed. Gene deadpanning when a rocket explodes, Gus wincing when the rover flips, Wernher treating explosions as data. This is gold but needs the core dialogue system in place first.

## Action Mode Radio Chatter
Mid-mission ambient callouts via Action presentation mode. Gene radio-calling when you hit orbit, Wernher chiming in on science collection, Gus grunting approval when the rover moves. Needs the core Tier 1/2 dialogues written first to establish character voices. Could integrate with BadgKatDialogue extension layer once built.

## Music Switcher Integration
Music Switcher is a recently released mod that allows config-driven music switching. Add custom music cues for BadgKatCareer moments — admin building scenes, milestone directives, act transitions, anomaly discoveries. Could pair specific tracks with specific characters or campaign acts.
